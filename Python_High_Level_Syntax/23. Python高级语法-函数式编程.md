## 1.3. 装饰器（decrator)
- 在不改动函数代码的基础上次无限扩展函数功能的一种机制
- 本质上讲，装饰器是一个返回函数的高阶函数
- 装饰器语法：
    - 使用@语法，即在每次要扩展到函数定义前使用@+函数名
    - 看实例23_2.py   
- 装饰器的好处：
    - 一旦定义，则可以装饰任意函数
    - 一旦被其装饰，则把装饰器的功能直接添加到定义的函数上
